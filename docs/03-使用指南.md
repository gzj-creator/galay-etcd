# 03-使用指南

本文档提供 Galay-Etcd 的详细使用说明、最佳实践和常见场景。

## 选择客户端类型

### 同步 vs 异步

| 特性 | 同步客户端 (EtcdClient) | 异步客户端 (AsyncEtcdClient) |
|------|------------------------|------------------------------|
| 适用场景 | 简单脚本、测试工具 | 高并发服务、异步应用 |
| 编程模型 | 阻塞调用 | 协程 + co_await |
| 性能 | 中等 | 高（非阻塞 IO） |
| 复杂度 | 简单 | 中等 |
| 依赖 | 无需 Runtime | 需要 IOScheduler |

### 使用建议

**选择同步客户端当**:
- 编写简单的配置管理工具
- 单次操作或低频调用
- 不需要高并发
- 快速原型开发

**选择异步客户端当**:
- 构建高性能服务
- 需要并发处理多个请求
- 与其他异步组件集成
- 需要精确的超时控制

## 同步客户端使用指南

### 基本连接与配置

```cpp
#include "galay-etcd/sync/EtcdClient.h"
#include <iostream>

using namespace galay::etcd;

int main() {
    // 方式 1：使用配置对象
    EtcdConfig config;
    config.endpoint = "http://127.0.0.1:2379";
    config.api_prefix = "/v3";
    config.request_timeout = std::chrono::seconds(5);
    config.buffer_size = 16384;
    config.keepalive = true;

    EtcdClient client(config);

    // 方式 2：使用默认配置
    EtcdClient client2;  // 默认连接 http://127.0.0.1:2379

    // 连接
    auto conn = client.connect();
    if (!conn) {
        std::cerr << "连接失败: " << conn.error().message() << '\n';
        return 1;
    }

    std::cout << "连接成功\n";
    return 0;
}
```

### KV 操作详解

#### 写入键值

```cpp
// 基本写入
auto put = client.put("/config/app/name", "my-app");
if (!put) {
    std::cerr << "写入失败: " << put.error().message() << '\n';
    return 1;
}

// 带租约的写入（键会在租约过期后自动删除）
auto lease = client.grantLease(30);  // 30 秒租约
if (lease) {
    int64_t lease_id = client.lastLeaseId();
    auto put = client.put("/temp/session", "active", lease_id);
    if (put) {
        std::cout << "临时键写入成功\n";
    }
}
```

#### 读取键值

```cpp
// 读取单个键
auto get = client.get("/config/app/name");
if (!get) {
    std::cerr << "读取失败: " << get.error().message() << '\n';
    return 1;
}

const auto& kvs = client.lastKeyValues();
if (kvs.empty()) {
    std::cout << "键不存在\n";
} else {
    for (const auto& kv : kvs) {
        std::cout << "键: " << kv.key << '\n';
        std::cout << "值: " << kv.value << '\n';
        std::cout << "版本: " << kv.version << '\n';
        std::cout << "创建版本: " << kv.create_revision << '\n';
        std::cout << "修改版本: " << kv.mod_revision << '\n';
    }
}
```

#### 前缀查询


### 租约管理

#### 创建和使用租约

```cpp
// 创建 60 秒租约
auto lease = client.grantLease(60);
if (!lease) {
    std::cerr << "创建租约失败: " << lease.error().message() << '\n';
    return 1;
}

int64_t lease_id = client.lastLeaseId();
std::cout << "租约 ID: " << lease_id << '\n';

// 绑定多个键到同一个租约
client.put("/session/user1", "online", lease_id);
client.put("/session/user2", "online", lease_id);
client.put("/session/user3", "online", lease_id);

// 所有键会在 60 秒后自动过期
```

#### 租约续期

```cpp
// 定期续约以保持键存活
while (running) {
    std::this_thread::sleep_for(std::chrono::seconds(20));

    auto keepalive = client.keepAliveOnce(lease_id);
    if (!keepalive) {
        std::cerr << "续约失败，重新创建租约\n";
        auto new_lease = client.grantLease(60);
        if (new_lease) {
            lease_id = client.lastLeaseId();
            // 重新注册键
            client.put("/session/user1", "online", lease_id);
        }
    } else {
        std::cout << "续约成功\n";
    }
}
```

### Pipeline 批量操作

```cpp
using PipelineOp = EtcdClient::PipelineOp;

// 构建批量操作
std::vector<PipelineOp> ops;
ops.push_back(PipelineOp::Put("/batch/key1", "value1"));
ops.push_back(PipelineOp::Put("/batch/key2", "value2"));
ops.push_back(PipelineOp::Put("/batch/key3", "value3"));
ops.push_back(PipelineOp::Get("/batch/", true));  // 前缀查询
ops.push_back(PipelineOp::Del("/batch/key2"));

// 执行批量操作
auto result = client.pipeline(std::move(ops));
if (!result) {
    std::cerr << "Pipeline 失败: " << result.error().message() << '\n';
    return 1;
}

// 检查每个操作的结果
const auto& results = client.lastPipelineResults();
for (size_t i = 0; i < results.size(); ++i) {
    const auto& r = results[i];
    std::cout << "操作 " << i << ": ";

    if (!r.ok) {
        std::cout << "失败\n";
        continue;
    }

    switch (r.type) {
        case EtcdClient::PipelineOpType::Put:
            std::cout << "写入成功\n";
            break;
        case EtcdClient::PipelineOpType::Get:
            std::cout << "查询成功，返回 " << r.kvs.size() << " 个键\n";
            for (const auto& kv : r.kvs) {
                std::cout << "  " << kv.key << " = " << kv.value << '\n';
            }
            break;
        case EtcdClient::PipelineOpType::Delete:
            std::cout << "删除成功，删除了 " << r.deleted_count << " 个键\n";
            break;
    }
}
```

## 异步客户端使用指南

### 基本设置

```cpp
#include "galay-etcd/async/AsyncEtcdClient.h"
#include <galay-kernel/kernel/Runtime.h>

using namespace galay::kernel;
using namespace galay::etcd;

Coroutine asyncExample(IOScheduler* scheduler) {
    // 配置超时
    AsyncEtcdConfig config;
    config.endpoint = "http://127.0.0.1:2379";
    config.request_timeout = std::chrono::milliseconds(5000);
    config.buffer_size = 32768;

    AsyncEtcdClient client(scheduler, config);

    // 连接
    auto conn = co_await client.connect();
    if (!conn) {
        std::cerr << "连接失败: " << conn.error().message() << '\n';
        co_return;
    }

    // 使用客户端...

    co_await client.close();
}

int main() {
    Runtime runtime(1, 1);
    runtime.start();

    auto* scheduler = runtime.getNextIOScheduler();
    std::atomic<bool> done{false};

    scheduler->spawn([&]() -> Coroutine {
        co_await asyncExample(scheduler);
        done.store(true);
    }());

    while (!done.load()) {
        std::this_thread::sleep_for(std::chrono::milliseconds(50));
    }

    runtime.stop();
    return 0;
}
```

## 实战场景

### 服务注册与发现

```cpp
// 服务注册
Coroutine registerService(IOScheduler* scheduler,
                          const std::string& service_name,
                          const std::string& service_addr,
                          int ttl_seconds) {
    AsyncEtcdClient client(scheduler);

    auto conn = co_await client.connect();
    if (!conn) {
        co_return;
    }

    // 创建租约
    auto lease = co_await client.grantLease(ttl_seconds);
    if (!lease) {
        std::cerr << "创建租约失败\n";
        co_await client.close();
        co_return;
    }

    int64_t lease_id = client.lastLeaseId();
    std::string key = "/services/" + service_name + "/" + service_addr;

    // 注册服务
    auto put = co_await client.put(key, service_addr, lease_id);
    if (!put) {
        std::cerr << "注册服务失败\n";
        co_await client.close();
        co_return;
    }

    std::cout << "服务注册成功: " << key << '\n';

    // 定期续约保持服务在线
    while (true) {
        co_await galay::kernel::sleep(std::chrono::seconds(ttl_seconds / 3));

        auto keepalive = co_await client.keepAliveOnce(lease_id);
        if (!keepalive) {
            std::cerr << "续约失败，重新注册\n";
            // 重新创建租约并注册
            lease = co_await client.grantLease(ttl_seconds);
            if (lease) {
                lease_id = client.lastLeaseId();
                co_await client.put(key, service_addr, lease_id);
            }
        }
    }

    co_await client.close();
}

// 服务发现
Coroutine discoverServices(IOScheduler* scheduler,
                           const std::string& service_name) {
    AsyncEtcdClient client(scheduler);

    auto conn = co_await client.connect();
    if (!conn) {
        co_return;
    }

    std::string prefix = "/services/" + service_name + "/";
    auto get = co_await client.get(prefix, true);

    if (!get) {
        std::cerr << "查询服务失败\n";
        co_await client.close();
        co_return;
    }

    std::cout << "发现服务实例:\n";
    for (const auto& kv : client.lastKeyValues()) {
        std::cout << "  地址: " << kv.value << '\n';
    }

    co_await client.close();
}
```

## 错误处理最佳实践

### 完整错误处理

```cpp
void handleEtcdError(const EtcdError& error) {
    std::cerr << "错误: " << error.message() << '\n';

    switch (error.type()) {
        case EtcdErrorType::Success:
            break;
        case EtcdErrorType::InvalidEndpoint:
            std::cerr << "端点配置错误，请检查 endpoint 格式\n";
            break;
        case EtcdErrorType::InvalidParam:
            std::cerr << "参数错误，请检查输入参数\n";
            break;
        case EtcdErrorType::NotConnected:
            std::cerr << "未连接，请先调用 connect()\n";
            break;
        case EtcdErrorType::Connection:
            std::cerr << "连接错误，请检查网络和 etcd 服务状态\n";
            break;
        case EtcdErrorType::Timeout:
            std::cerr << "请求超时，请检查网络延迟或增加超时时间\n";
            break;
        case EtcdErrorType::Send:
            std::cerr << "发送数据失败\n";
            break;
        case EtcdErrorType::Recv:
            std::cerr << "接收数据失败\n";
            break;
        case EtcdErrorType::Http:
            std::cerr << "HTTP 协议错误\n";
            break;
        case EtcdErrorType::Server:
            std::cerr << "etcd 服务端错误\n";
            break;
        case EtcdErrorType::Parse:
            std::cerr << "响应解析错误\n";
            break;
        case EtcdErrorType::Internal:
            std::cerr << "内部错误\n";
            break;
    }
}

// 使用示例
auto result = client.put("/key", "value");
if (!result) {
    handleEtcdError(result.error());
    return 1;
}
```

### 重试机制

```cpp
template<typename Func>
auto retryOperation(Func&& func, int max_retries = 3) {
    for (int i = 0; i < max_retries; ++i) {
        auto result = func();
        if (result) {
            return result;
        }

        std::cerr << "操作失败，重试 " << (i + 1) << "/" << max_retries << '\n';
        std::this_thread::sleep_for(std::chrono::seconds(1));
    }

    return func();  // 最后一次尝试
}

// 使用示例
auto result = retryOperation([&]() {
    return client.put("/key", "value");
});
```

## 性能优化建议

### 1. 连接复用

```cpp
// 好：复用连接
EtcdClient client;
client.connect();

for (int i = 0; i < 1000; ++i) {
    client.put("/key" + std::to_string(i), "value");
}

client.close();

// 不好：频繁创建连接
for (int i = 0; i < 1000; ++i) {
    EtcdClient client;
    client.connect();
    client.put("/key" + std::to_string(i), "value");
    client.close();
}
```

### 2. 使用 Pipeline 批量操作

```cpp
// 好：使用 Pipeline
std::vector<EtcdClient::PipelineOp> ops;
for (int i = 0; i < 100; ++i) {
    ops.push_back(EtcdClient::PipelineOp::Put(
        "/key" + std::to_string(i), "value"));

client.pipeline(std::move(ops));

// 不好：逐个操作
for (int i = 0; i < 100; ++i) {
    client.put("/key" + std::to_string(i), "value");
}
```

### 3. 合理设置超时

## 常见问题

### Q: 如何处理连接断开？

A: 检测连接状态并重连：

```cpp
if (!client.connected()) {
    auto conn = client.connect();
    if (!conn) {
        std::cerr << "重连失败\n";
        return 1;
    }
}
```

### Q: 租约过期后键会立即删除吗？

A: 是的，租约过期后，绑定到该租约的所有键会被 etcd 自动删除。建议定期续约以保持键存活。

### Q: Pipeline 操作是原子的吗？

A: 不是。Pipeline 只是批量发送多个操作，每个操作独立执行，不保证原子性。

### Q: 如何实现配置热更新？

A: 使用 etcd 的 Watch 功能（当前版本未实现），或定期轮询配置键：

```cpp
while (running) {
    auto get = client.get("/config/", true);
    if (get) {
        // 更新本地配置
        updateLocalConfig(client.lastKeyValues());
    }
    std::this_thread::sleep_for(std::chrono::seconds(5));
}
```

### Q: 同步客户端和异步客户端可以混用吗？

A: 可以，但它们是独立的连接。建议在同一个应用中统一使用一种客户端类型。
